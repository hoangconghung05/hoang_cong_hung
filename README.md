# Trang giới thiệu Quảng Trị & Quảng Bình (demo)

Trang web tĩnh demo để giới thiệu danh lam thắng cảnh và di tích lịch sử của Quảng Trị và Quảng Bình (theo yêu cầu). Nội dung bằng tiếng Việt, có ảnh và video minh họa.

Files:

- `index.html` — trang chính (HTML)
- `css/style.css` — stylesheet
- `js/script.js` — script cho menu, gallery modal, smooth scroll

Chạy (mở trang):

1. Mở `index.html` trong trình duyệt (double-click hoặc chuột phải → Open with...).

Ghi chú:

- Ảnh sử dụng API công khai Unsplash (placeholder). Thay bằng ảnh nội bộ trong `assets/` nếu cần.
- Video là iframe YouTube search playlist (placeholder). Thay `src` iframe bằng URL embed của video cụ thể khi có.
- Nếu muốn đóng gói ảnh/video offline, thêm thư mục `assets/` và cập nhật đường dẫn trong `index.html`.

Nếu bạn muốn tôi thêm nội dung chi tiết cho từng địa điểm (bản đồ, lịch sử đầy đủ, tuyến du lịch đề xuất), tôi có thể mở rộng trang này.
